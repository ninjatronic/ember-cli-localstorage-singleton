# Contribution Guidelines

This repository uses [git-flow](https://github.com/nvie/gitflow). Please use the
standard naming conventions. Commit messages are written in sentence case and
lines are <= 80 characters long.

## Issues

Please consider providing the following with your issue report:

* Output from the debug console
* Steps to reproduce
* A clear and concise explanation of the problem
* A JSBin or JSFiddle reproducing the problem

## Pull Requests

Pull requests are welcome. Please craft your pull request from an appropriately
named branch.
